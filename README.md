# Sombre — 마농 브레슈 × 딜런 토리 섬너

> 가슴에 손을 얹어. ——심장이 뛰나?  
> ⋯그래. 뛰고 있어.

## 개요

호그와트 마법학교를 배경으로 한 마농 브레슈와 딜런 토리 섬너의 이야기를 담은 웹사이트입니다.

## 캐릭터

- **현직**: 지팡이 장인 견습생

## 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Database**: Vercel Postgres
- **Font**: Pretendard, Cinzel, Gowun Batang, Space Mono, Jeju Myeongjo

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 환경 변수

```env
DATABASE_URL=your_database_url
```

## 페이지 구성

- **Home** (`/`): 메인 페이지, INFO 및 PLAYER 섹션
- **Characters** (`/character`): 마농/딜런 캐릭터 프로필
- **Timeline** (`/timeline`): 연대기
- **Record** (`/record`): 역극 및 TRPG 기록
- **Admin** (`/admin`): 관리자 페이지 (비밀번호: sombre1234)
