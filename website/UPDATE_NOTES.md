# 🎭 SAME Admin 페이지 업데이트

## ✨ 새로운 기능

### 1. 📁 txt 파일 병합 기능
- ✅ **새 파일로 시작**: 새로운 레코드 생성
- ✅ **현재 레코드에 추가**: 기존 레코드에 txt 파일 내용 추가
- ✅ 여러 txt 파일을 하나의 레코드로 통합 가능

### 2. 🖼️ 이미지 업로드 기능
- ✅ **말풍선에 이미지 추가**: 각 대사에 이미지 첨부 가능
- ✅ **이미지 미리보기**: Admin 페이지에서 바로 확인
- ✅ **이미지 삭제**: X 버튼으로 이미지 제거
- ✅ **Base64 저장**: 별도 이미지 서버 없이 데이터에 포함

### 3. 🖼️ Record 페이지 이미지 갤러리
- ✅ **이미지 클릭**: 전체화면으로 확대
- ✅ **좌우 화살표**: 다른 이미지로 이동
- ✅ **키보드 조작**: ← → 키로 이동, ESC로 닫기
- ✅ **이미지 정보**: 화자 이름과 현재 위치 표시

### 4. 차수(Phase) 관리
- ✅ **차수 추가**: 좌측 네비게이션 하단의 "+ 차수 추가" 버튼
- ✅ **차수 삭제**: 각 차수 옆 🗑️ 버튼
- ✅ **차수 이름 수정**: 차수 이름을 더블클릭하면 편집 가능
- ✅ **차수 순서 변경**: ⣿ 아이콘을 드래그하여 순서 변경

### 5. 소제목(Section) 관리
- ✅ **소제목 추가**: 각 차수 하단의 "+ 소제목 추가" 버튼
- ✅ **소제목 삭제**: 각 소제목 옆 ✕ 버튼 (마지막 소제목은 삭제 불가)
- ✅ **소제목 이름 수정**: 소제목을 더블클릭하면 편집 가능
- ✅ **소제목 순서 변경**: ⣿ 아이콘을 드래그하여 순서 변경

### 6. 기존 기능 유지
- ✅ txt 파일 업로드 및 자동 파싱
- ✅ 대사 추가/수정/삭제
- ✅ 대사 순서 드래그 앤 드롭
- ✅ 화자 및 대사 내용 수정

## 📝 사용 방법

### 🆕 txt 파일 병합하기
1. **새 레코드 생성**: 좌측 상단 "새 파일로 시작" 버튼으로 txt 파일 업로드
2. **기존 레코드에 추가**: 레코드 선택 후 "현재 레코드에 추가" 버튼으로 txt 파일 추가
3. 여러 파일을 순차적으로 추가하여 하나의 레코드로 병합 가능

### 🖼️ 이미지 추가하기
1. 대사를 선택하고 **"📷 이미지 추가"** 버튼 클릭
2. 이미지 파일 선택 (jpg, png, gif 등)
3. 이미지가 말풍선에 표시되며, X 버튼으로 삭제 가능
4. Record 페이지에서 이미지 클릭 시 전체화면 갤러리로 확대

### 차수 추가하기
1. 좌측 네비게이션 하단의 **"+ 차수 추가"** 버튼 클릭
2. 새로운 차수가 생성됩니다 (기본 이름: "n차")
3. 차수 이름을 더블클릭하여 원하는 이름으로 변경

### 소제목 추가하기
1. 원하는 차수 하단의 **"+ 소제목 추가"** 버튼 클릭
2. 새로운 소제목이 생성됩니다 (기본 이름: "새 소제목")
3. 소제목을 더블클릭하여 원하는 이름으로 변경

### 순서 변경하기
1. 차수 또는 소제목 왼쪽의 **⣿ 아이콘**을 드래그
2. 원하는 위치에 드롭하여 순서 변경

### 삭제하기
- **차수 삭제**: 차수 옆 🗑️ 버튼 클릭
- **소제목 삭제**: 소제목 옆 ✕ 버튼 클릭 (각 차수는 최소 1개의 소제목 필요)

### 이름 수정하기
1. 차수 또는 소제목을 **더블클릭**
2. 편집 모드로 전환되면 원하는 이름 입력
3. Enter 키 또는 다른 곳 클릭으로 저장

```

✅ 차수/소제목 자유롭게 추가
✅ 드래그로 순서 변경
✅ 더블클릭으로 이름 수정
✅ 필요한 차수/소제목만 선택적 삭제
✅ 이미지 업로드 및 갤러리 뷰어
✅ 여러 txt 파일을 하나의 레코드로 병합
✅ Base64로 이미지 저장 (서버 불필요)
```

## 💡 팁

1. **빠른 편집**: 차수/소제목 이름을 더블클릭하면 바로 수정 가능
2. **순서 조정**: ⣿ 아이콘을 드래그하면 자유롭게 순서 변경
3. **저장 필수**: 모든 변경사항은 "저장하기" 버튼을 눌러야 반영됩니다
4. **실시간 미리보기**: 좌측에서 차수/소제목을 클릭하면 우측에서 내용 확인
5. **이미지 크기**: 큰 이미지는 자동으로 압축되지 않으므로 적절한 크기로 업로드 권장
6. **txt 병합**: 여러 대화 파일을 하나의 레코드로 관리할 때 유용
7. **갤러리 네비게이션**: 이미지 클릭 후 키보드 화살표(← →)로 빠른 이동
